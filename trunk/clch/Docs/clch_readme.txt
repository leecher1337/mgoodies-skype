Contact Left Channel History plugin
-----------------------------------

CAUTION: THIS IS AN ALPHA STAGE PLUGIN. IT CAN DO VERY BAD THINGS. USE AT YOUR OWN RISK.

This is a very simple plugin that logs to the history when a contact left a chanel and this history window is open. The logged entry has the type EVENTTYPE_CONTACTLEFTCHANNEL (9004), and, by now, only tabSRMM and History++ are able to display it in the message log.

It requires a modified MSN version to work. It can be found here:
Unicode: http://pescuma.mirandaim.ru/miranda/msnW.clch.zip
ASCII: http://pescuma.mirandaim.ru/miranda/msn.clch.zip
Patch: http://pescuma.mirandaim.ru/miranda/msn.clch.patch
