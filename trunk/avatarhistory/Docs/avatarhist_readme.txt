Avatar History plugin
-----------------------------------

CAUTION: THIS IS AN ALPHA STAGE PLUGIN. IT CAN DO VERY BAD THINGS. USE AT YOUR OWN RISK.

This plugins keep a history of contact avatars. It stores the files on the disk and uses
miranda history to keep track of then.

This plugin is really made by MattJ, I'm only doing some changes to allow better handling
of the avatars.

To report bugs/make suggestions, go to the forum thread: http://forums.miranda-im.org/showthread.php?t=9118
