History Keeper plugin
---------------------

CAUTION: THIS IS ALPHA QUALITY SOFTWARE. USE AT YOUR OWN RISK.

This plugin logs some events to the history and show popups. 

The events that are handled are:
 - Nickname changes
 - Status Message changes
 - Client changes
 - Status changes

For it to work, it depends on some popup plugin (YAPP or Popup+) and on History Events plugin: http://pescuma.mirandaim.ru/miranda/historyevents

This plugin requires at least Miranda 0.7

The icons were copied/based on the icons from History++.
